The output of this project is the unit package file file Environment.unitypackage.
The package is sourced from the Assets/Environment directory.

This package contains trees, details, terrain layers water elements curated from the Unity Standard Assets 2018 and Unity Terrain Sample Assets Pack packages.
Most of the original .tif textures have been replaced with .png textures to reduce file size.  (The package is approximately 3X larger with .tif files).
